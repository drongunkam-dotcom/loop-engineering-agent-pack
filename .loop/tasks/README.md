# Tasks

This directory stores Loop Engineering tasks when the pack is used inside a project.

Expected task folder shape:

```text
task-XXX-short-slug/
  contract.md
  state.md
  maker-report.md
  checker-report.md
  architecture-report.md
  handoff.md
```

In this repository, this folder is mostly a reference placeholder. In target projects, `loop_planner` and `loop_contract_writer` create real task folders here.
