# Loop Engineering Agent Pack

Русскоязычный пакет для Codex.

## Установка

```bash
npx loop-engineering-agent-pack install
```

## Что ставится

```text
~/.codex/agents/loop-orchestrator.toml
```

## Работа

Открой проект в Codex Desktop и используй `loop_orchestrator`.

Документация: `docs/NPM_USAGE_RU.md`.
